Créditos

Principais tecnologias que foram utilizadas:
Play framework: tecnologia utilizada como Server Side.
Twitter bootstrap: biblioteca utilizada principalmente para estilizar os html's das páginas.
AngularJS: biblioteca JS que possibilitou o MVC no FrontEnd.

Outras tecnologias utilizadas:
JQuery: biblioteca JS para manipulação de DOM.
Hibernate: ORM utilizado para abstração da camada de armazenamento de dados.
MySQL Server: Banco de dados utilizados para o armazenamento dos dados.
Eclipse: IDE utilizada no desenvolvimento.
GIT: Controle de versão utilizado para o versionamento do código.
GSON: biblioteca da Google utilizada para a serialização e deserialização de objetos de dados.
